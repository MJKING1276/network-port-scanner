# Network Port Scanner Using Python

This project is a GUI-based Network Port Scanner developed using Python and Tkinter. It allows users to scan a target system and identify open ports efficiently.

## Features

* User-friendly graphical interface
* Multi-threaded port scanning for faster performance
* Real-time display of open ports
* Progress tracking and elapsed time
* Option to save scan results

## Technologies Used

* Python
* Tkinter
* Socket
* Threading
* Queue

## How to Run

1. Run the Python file:
   python port_scanner.py

2. Enter target IP and port range

3. Click "Start Scan"

## Output

Displays open ports along with their respective services
